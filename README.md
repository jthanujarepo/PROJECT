## Online Bookstore 

**Project Summary**

 The Online Bookstore is a web-based application developed using Python and Django that allows users to browse books, manage a shopping cart, and perform user authentication. 
 The project demonstrates backend development skills, database integration, and MVC/MVT architecture.
